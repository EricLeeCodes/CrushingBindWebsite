# CrushingBindWebsite

Crushing Bind is a Web Novel website where the administrator can Create, Read, Update, and Delete each page of their website using QuillJS as a writing platform which can be directly displayed on the public site.

The administrator page is locked behind a Sign In page where the admin is already authenticated and authorized.
